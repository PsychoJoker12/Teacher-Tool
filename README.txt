This is a Python program designed for teachers to easily make multiple choice tests and multiple forms and export them to Microsoft Word documents (.docx)
In order for this program to function, you will need to install the python-docx library which can be found here:
    https://python-docx.readthedocs.io/en/latest/

    To install python-docx, execute "python -m pip install python-docx" in the command prompt/terminal

To run Teacher Tool, execute "TeacherTool.py"
